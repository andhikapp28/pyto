# Plan: Tutorial Python untuk Semua Umur

**Target:** Anak-anak, remaja, dan dewasa non-IT (materi berlapis)
**Format:** Buku (PDF/Word) + Web interaktif + Konten pendek
**Cakupan:** Python dasar
**Prinsip:** Satu sumber materi ("single source"), lalu diolah jadi 3 format.
**Fokus utama:** Bikin Python menyenangkan dan mudah dipahami — kesederhanaan di atas kelengkapan.

**Keputusan PDF:**
- PDF dicetak **per bab** (1 bab = 1 file), dipakai sebagai lampiran unduhan di web interaktif.
- Penomoran halaman tiap PDF **mulai dari 1** sendiri.
- PDF gabungan (semua bab jadi satu) **ditunda dulu** — dikerjakan nanti.

---

## Konsep Besar

Kita pakai satu maskot/cerita pemandu (misal robot bernama "Pyto") yang menemani pembaca dari bab ke bab. Setiap konsep dijelaskan dengan: analogi dunia nyata → contoh kode super pendek → coba sendiri. Ini membuat materi tetap satu gaya di semua format.

**Sistem berlapis** biar cocok anak sampai dewasa:
- **Kotak Inti** — penjelasan utama, bahasa sesederhana mungkin (untuk semua).
- **Kotak "Main Yuk"** — aktivitas/analogi visual untuk anak.
- **Kotak "Tahu Lebih"** — sedikit tambahan untuk remaja/dewasa yang penasaran.

---

## Kurikulum (10 Bab Dasar)

1. **Kenalan dengan Python** — apa itu coding, kenapa seru, cara jalankan kode.
2. **Menyapa Dunia (`print`)** — menampilkan tulisan, si komputer "ngomong".
3. **Kotak Ajaib (Variabel)** — menyimpan angka, nama, apa saja.
4. **Ngobrol dengan Komputer (`input`)** — komputer bertanya, kita jawab.
5. **Angka & Hitung-hitungan** — kalkulator sederhana.
6. **Kalau Begini, Maka Begitu (`if`)** — komputer mengambil keputusan.
7. **Ulang-ulang (Loop)** — menyuruh komputer mengulang tanpa capek.
8. **Kumpulan Barang (List sederhana)** — daftar teman, daftar belanja.
9. **Salah itu Wajar (Error & memperbaiki)** — cara membaca pesan error tanpa panik.
10. **Proyek Seru** — 3 mini-project: Tebak Angka, Kuis, Kalkulator.

Setiap bab: 1 konsep + 1 contoh + 1 latihan kecil. Pendek dan bertahap.

---

## Rencana per Format

### 1. Buku (PDF per bab)
- Tiap bab jadi **1 file PDF terpisah**, penomoran halaman mulai dari 1.
- Struktur tiap bab: judul bab → konsep inti → contoh → "Main Yuk" → latihan kecil.
- Tiap bab 3-5 halaman, banyak ruang kosong + ilustrasi/diagram sederhana.
- Output: `.docx` (mudah diedit) → ekspor ke `.pdf` per bab, dipakai sebagai lampiran unduhan di web.
- *(PDF gabungan semua bab: nanti.)*

### 2. Web Interaktif
- Website satu halaman per bab, navigasi sidebar.
- Fitur andalan: **area coba kode langsung di browser** (pakai editor Python berbasis web) supaya pembaca tak perlu instal apa pun.
- Ringan, bisa dibuka di HP. Output: file HTML sederhana.

### 3. Konten Pendek
- Tiap konsep jadi 1 "kartu" (slide/gambar) — cocok untuk Instagram/TikTok/WA.
- Format: 1 analogi + 1 potongan kode + 1 hasil. Total ~10-15 kartu.
- Output: kumpulan slide (bisa `.pptx` atau gambar).

## Urutan Pengerjaan (Bertahap)

1. **Finalisasi kurikulum & gaya** — kunci nama maskot, gaya bahasa, contoh. *(fondasi semua format)*
2. **Tulis naskah master** — 10 bab dalam 1 file markdown sebagai sumber tunggal.
3. **Buat Buku** — olah naskah jadi `.docx` + `.pdf` berilustrasi.
4. **Buat Web** — ubah naskah jadi web interaktif dengan editor kode.
5. **Buat Konten Pendek** — ringkas tiap bab jadi kartu-kartu.
6. **Review & uji** — cek semua kode benar-benar jalan, bahasa mudah dipahami.
